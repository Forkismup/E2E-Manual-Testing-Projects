# E2E Manual Testing Projects

### Complete End-to-End Web and Mobile App Manual Testing Portfolio

This repository contains production-grade manual testing documentation for multiple real-world applications including e-commerce platforms, mobile learning apps, and return management systems.
Each project includes end-to-end test scenarios, detailed test cases, defect reporting, and full coverage documentation following ISTQB standards.

---

## Repository Highlights

| Feature                    | Description                                                                             |
| -------------------------- | --------------------------------------------------------------------------------------- |
| End-to-End Manual Testing  | Covers complete user journeys such as login, purchase, checkout, and profile management |
| Multi-Domain Coverage      | E-commerce, mobile learning apps, logistics systems                                     |
| Structured Test Cases      | Includes steps, data, expected results, status, severity, and priority                  |
| Test Design Techniques     | Boundary value, equivalence partitioning, negative testing                              |

| Professional Documentation | Follows industry-standard testing formats                                               |

## Documentation Included in Each Project

### 1. Test Scenarios

High-level test coverage, grouped feature-wise with positive and negative flows.

### 2. Detailed Test Cases

Includes:

* Test case ID
* Description
* Preconditions
* Test steps
* Test data
* Expected and actual results
* Execution status
* Severity and priority
* Comments

### 3. Bug Report (If applicable)

Includes:

* Bug ID
* Steps to reproduce
* Expected vs actual result
* Environment information
* Screenshots reference

### 4. End-to-End Workflow Coverage

Examples:

* Login to checkout process
* Signup to profile update flow
* Navigation testing
* Product ordering and tracking

## Testing Techniques Used

| Technique                 | Purpose                                    |
| ------------------------- | ------------------------------------------ |
| Functional Testing        | Core application functionality             |
| Regression Testing        | Ensuring existing features remain stable   |
| Exploratory Testing       | Identifying hidden defects                 |
| UI/UX Testing             | Checking layout, style, and responsiveness |
| Negative Testing          | Invalid input and failure conditions       |
| Boundary Testing          | Max/min input values                       |
| Cross-Platform Validation | Web vs mobile behavior                     |
